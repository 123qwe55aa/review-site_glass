# 玻璃工艺学复习网站

这是一个无需安装依赖的静态复习网站，内容按 9 个章节整理，支持知识点浏览、分章节背题、乱序复习、待回炉题目、搜索和本地进度保存。

在仓库根目录启动：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173/review-site/
```

背题进度保存在当前浏览器的 `localStorage` 中。PDF 课件为图片型文件，网站中的知识框架按课件主题归类，重点问答主要整理自《无机221-胡锡鸿.docx》。

`pdfs/` 目录中包含用于网页快速查阅的压缩版课件；原始高清课件仍保留在仓库的 `glass_Tech/` 目录中。

题库中标有“课件摘取 · … P页码”的题目，来自对应后半章节课件页面；未标注“课件摘取”的重点问答主要来自《无机221-胡锡鸿.docx》。
